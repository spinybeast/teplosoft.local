Call me button module

This module will automatically add a call me button to your Prestashop template.

Enter your phone number and increase customers confidence with this easy to use button.

If you can not see the button in mobile devices, activate "Enable Mobile" under Prestashop -> Installed Modules (see screenshot below).

ENABLE-MOBILE-SETTINGS.PNG

Features:

- Select button circle radius.
- Select icon color.
- Select background color.
- Select margin.
- Align bottom left or bottom right.
- Enable or disable in desktop devices.
